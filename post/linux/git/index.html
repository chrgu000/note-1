1. git

1.1. 命令

1.1.1. 远程分支

  1. 查看远程分支

$ git remote -v
origin  git@git.oschina.net:dishui/local_b2b.git (fetch)
origin  git@git.oschina.net:dishui/local_b2b.git (push)

  1. 添加远程分支

git remote add origin git@git.oschina.net:dishui/local_b2b.git

  1. 设置远程分支

git remote set-url origin git@github.com:dishuiGit/SublimeNote.git

  1. 添加

git add .		#添加
git reset .		#撤销添加

  1. 更新

git pull origin master

  1. 查看是否有文件需要提交
