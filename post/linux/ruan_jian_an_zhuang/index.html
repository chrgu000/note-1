1. RPM 安装软件

  1. 指定安装目录

rpm -ivh --prefix=/opt/temp  xxx.rpm

2. 安装 Java

  1. 查看 Linux 自带的 JDK 是否已安装

java -version

ps:如果出现 openjdk ，最好还是先卸载掉 openjdk ,在安装 sun 公司的 jdk .

  1. 查看 jdk 安装信息

rpm -qa|grep java
rpm -qa|grep jdk

  1. 卸载`OpenJDK` ，执行以下操作

rpm -e --nodeps tzdata-java-2012c-1.el6.noarch
rpm -e --nodeps java-1.7.0-openjdk-1.7.0.45-1.45.1.11.1.el6.x86_64

  1. 新建 java 安装目录

mkdir /usr/java

  1. 将之前下载的 jdk 解压缩并安装

=源码
tar -zxvf  jdk-7u71-linux-i586.tar.gz
=rpm
rpm -ivh jdk-7u79-linux-i586.rpm --prefix=/usr/java/

  1. 在 profile 文件中加入 java 环境变量

vi /etc/profile
=
export JAVA_HOME=/usr/java/jdk1.7.0_71
export CLASSPATH=.:%JAVA_HOME%/lib/dt.jar:%JAVA_HOME%/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin

  1. 使文件立即生效

source /etc/profile

3. Vanish

  1. vanish 4.1

yum install epel-release
rpm --nosignature -i https://repo.varnish-cache.org/redhat/varnish-4.1.el7.rpm
(or: rpm --nosignature -i https://repo.varnish-cache.org/redhat/varnish-4.1.el6.rpm for RHEL6)
yum install varnish

[[Installation on RedHat|http://www.varnish-cache.org/releases/install_redhat.html]]

4. Python3

  1. CentOS7 编译安装

yum groupinstall "Development Tools" &&\
yum install openssl-devel
tar -zxf Python-3.5.2.tgz
cd Python-3.5.2
./configure --prefix=/usr/local/python3/
make && make install
# 链接
ln -s /usr/local/python3/bin/python3 /usr/bin/python3
