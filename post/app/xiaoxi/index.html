14.1添加好友列表

URL

http://116.90.81.68/app/app_message/addFriendList.shtml

特殊说明
支持格式

JSON

HTTP请求方式

GET

HTTP请求方式

Content-Type: application/json;charset=UTF-8

携带参数说明
{
  "userId" : "20150828QYY07uh"//当前用户ID
}
是否需要登录

是

成功
{
"list":[
{
"id":"20160817K5r07Pt", //主键ID
"addStatus":1, //添加状态：0已申请，1已添加，2已拒绝
"inDate":"2016-08-17 09:30:30", //时间
"addedUserId":"201608153822qIE", //被添加用户ID
"userId":"20150828p8jr40S", //用户ID
"image":"", //申请人头像
"userName":"ceshi_buyer", //用户名称
"addedUserName":"bqlq" //被添加用户名称
}
]
"state":1 //1:成功 0:失败
}
失败
{
	"state" : 0, //1:成功 0:失败
	"message" : "获取列表失败" //state为0时需返回的错误说明信息
}
