采购单列表

URL

http://app.mailiqing.com/app/procurement/list?start=1&limit=10&area=北京&sort=1&q=沥青

支持格式

GET JSON

HTTP请求方式

Content-Type: application/json;charset=UTF-8

携带参数说明
start # 起始页
limit # 每页个数
area # 地区筛选
sort # 日期排序 0:降序 1:升序
综合排序 传 start limit
q # 搜索关键字
