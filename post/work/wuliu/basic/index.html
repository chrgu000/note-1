4. Q & A

Exception: java.lang.OutOfMemoryError

  1. windows 下

    set JAVA_OPTS=-Xms1024m -Xmx4096m -XX:PermSize=128m -XX:MaxPermSize=256m

  2. linux 下

    CATALINA_OPTS="-Xms2048m -Xmx2048m -XX:PermSize=512M -XX:MaxNewSize=256m -XX:MaxPermSize=1024m"
