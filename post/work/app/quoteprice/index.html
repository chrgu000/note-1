1. 报价单

1.1. 报价单发布

URL

http://116.90.81.68/app/quoteprice/add

支持格式

POST JSON

HTTP请求方式

Content-Type: application/json;charset=UTF-8

携带参数说明
{
  "userId" : "201512116eSr5i4" # 用户id
  "procurId" : "20160928TyWUN1L", # 采购单id
  "factoryName" : "兆华", # 公司名称
  "repositoryAddr" : "北京|北京市", # 出库地址
  "desc" : "70#沥青好东西", # 报价描述
  "contact" : "小明", # 联系人
  "phone" : "13345431231", # 电话
  "img" : "img/asdlfjl13345431231.img" # 图片 多张使用 | 拼接
}
是否需要登录

是

成功
{
  "state" : 1,
  "id" : "201609281332179" # 报价单 id
}
失败
{
  "state" : 0,
  "message" : "添加报价单失败！"
}
