1. 四大法则

\[[u(x) \pm v(x)]' = u'(x) + v'(x)\]
\[[u(x)v(x)]' = u'(x)v(x) + u(x)v'(x)\]
\[[\frac{u(x)}{v(x)}]' = \frac{u'(x)v(x)-u(x)v'(x)}{v^2(x)}'\]
\[[\frac{1}{v(x)}]' = - \frac{v'(x)}{v^2(x)}\]
\[[u \pm v]' = u' \pm v'\]
\[(uv)' = u'v + uv'\]
\[(\frac{u}{v})' = \frac{u'v-uv'}{v^2}\]

推广

\[[u + v + w]' = u' + v' + w'\]
\[(uvw)' = u'vw + uv'w + uvw'\]
\[\displaystyle \sum_{i=1}^n C_{i}u_{i})' = \sum C_{i}u_{i}'\]

线性组合的导数 = 导数的线性组合

\[\displaystyle \prod_{i=1}^{n}u_i)' = \sum_{i=1}^{n}u_1{\ldots}u_i'{\ldots}u_n\]

六个三角函数的导数

\[\displaystyle (\sin x)' = \cos x \qquad (\cos x)' = - \sin x \\[2ex] (\tan x)' = \sec^2 x \qquad (\cot x)' = - \csc^2 x \\[2ex] (\sec x)' = \sec x \tan x \qquad (\csc x)' = - \csc x \cot x\]

2. 函数求导法则2

例子
\[y= \sin 2x\]
\[\frac{d}{dx}(\sin x) = \cos x\]
符合函数求导
\[\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}\]
定理3 (复合函数的求导法则)

> 设函数 \(u=g(x)\) 在点x可导,而函数 \(y=f(u)\) 在点 \(u=g(x)\) 可导, 则复合函数 \(y=f(g(x))\) 在点 x 可导,且有导数

\[\displaystyle \frac{dy}{dx} = f'(u)g'(x)\]
\[\displaystyle \frac{dy}{dx} = \frac{dy}{du} \frac{du}{dx}\]
链式法则
\[\displaystyle x \rightarrow^g u \rightarrow^f y\]
