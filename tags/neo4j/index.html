Contents 1. 参考 例如, 一个可以用来查找 Emil 的朋友的模式: MATCH (ee:Person)-[:KNOWS]-(friends) WHERE ee.name = "Emil" RETURN ee, friends MATCH用于描述由已知节点查找节点的模式的子句 (ee)以 Person 开始模式 (由 WHERE 限制) -[:KNOWS]-匹配 "KNOWS" 关系 (任意方向) (friends)将被绑定到 Emil 的朋友 模式匹配可用于做推荐.
